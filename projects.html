Image-to-STL Heightmap Generator

JavaScript Canvas API Computational Geometry 3D Printing

Problem

Converting 2D images into 3D printable models typically requires expensive software or cloud services with privacy concerns. Users need a fast, client-side solution for creating heightmap-based 3D models from logos, artwork, or photographs.

Solution

Built a browser-based converter that transforms any image into an STL file with customizable parameters. The tool uses Canvas API for image processing, applies grayscale-to-height mapping with configurable curves, and generates ASCII STL format with proper face normals for 3D printing.

Technical Highlights

  • 100% Client-Side: No server required, all processing in-browser
  • Parametric Control: Base dimensions, extrude height, thickness, quality settings
  • Height Curves: Linear, sqrt, and quadratic mapping for material optimization
  • STL Generation: Algorithmic mesh creation with proper topology
  • Real-time Preview: Instant visual feedback before generation

Impact

Enables rapid prototyping of embossed parts without external tools. Useful for creating custom nameplates, signage, decorative panels, and relief sculptures.

Interactive 3D Model Portfolio

Three.js WebGL STLLoader Responsive Design

Problem

Showcasing 3D printable models on the web requires users to download files and open them in specialized software. This creates friction and limits engagement with a portfolio.

Solution

Developed a full-featured 3D viewer using Three.js that loads and displays STL files directly in the browser. Features orbit controls, color customization, dynamic lighting, and automatic camera positioning based on model bounds.

Technical Highlights

  • STL Parsing: Binary and ASCII STL format support
  • Auto-Centering: Bounding box calculation for optimal view
  • Material System: Real-time color changes with Phong shading
  • Multi-Source Loading: File upload and library selection
  • Performance: Efficient geometry disposal and memory management

Impact

Provides instant 3D visualization without downloads. Visitors can explore models, check dimensions, and assess print feasibility directly in browser.

LLM Evaluation & Quality Assurance

AI/ML Model Auditing Rubric Design Data Quality

Problem

Solution

Impact
